## Типографіка

**Принцип:** Мінімалізм + читабельність. Жодних декоративних шрифтів.

| Рівень | Шрифт | Вага | Розмір (desktop) |
|--------|-------|------|-----------------|
| H1 Hero | Georgia / serif | 300 (light) | 56–72px |
| H2 Секції | Inter / system-ui | 400 | 36–44px |
| H3 Картки | Inter / system-ui | 500 | 20–24px |
| Body | Inter / system-ui | 400 | 16px |
| Small / Caption | Inter / system-ui | 400 | 13–14px |
| Badge / Label | Inter / system-ui | 600 uppercase | 11–12px |

**Міжрядковий інтервал:** 1.6 для body, 1.2 для заголовків  
**Трекінг заголовків H1:** `tracking-tight` або `-0.02em`  

---

## Іконографія

**Стиль:** Лінійні іконки (stroke, 1.5–2px), без заливки, округлі кути  
**Розмір у блоках переваг:** 40×40px  
**Колір іконок:** Primary `#C4704F` або Text Secondary `#6B5B4E`  
**Джерела:** Heroicons (безкоштовні SVG), інлайн у JSX через `<svg>`

## Компоненти UI

### Кнопки

**Primary CTA:**
- Фон: `#C4704F` → hover `#A85A3A`
- Текст: білий, 14px, uppercase, letter-spacing 0.05em
- Padding: 14px 32px
- Border-radius: `rounded-none` (прямокутник — відчуття ремісника)
- Transition: 200ms

**Secondary / Ghost:**
- Border: 1.5px `#C4704F`
- Текст: `#C4704F`
- Hover: заливка primary

**Add-to-cart (картка товару):**
- Компактний, ширина 100%, border-radius `rounded-sm`
- Іконка кошика + текст «В кошик»

### Картки товарів
- Фон: `#FFFFFF`
- Shadow: `shadow-sm` → hover `shadow-md`
- Border: `border border-[#E8DDD4]`
- Image aspect ratio: `aspect-[4/3]`
- Hover: `scale(1.01)` на зображенні
- Бейджі: абсолютне позиціонування, top-left

### Бейджі
- «Акція»: фон `#6B8F71`, текст білий
- «Новинка»: фон `#C8A86B`, текст білий
- Розмір: text-[11px] font-semibold uppercase px-2 py-0.5 rounded-sm

### Форми / інпути
- Border: `1px solid #E8DDD4`
- Focus: `border-[#C4704F] outline-none ring-0`
- Border-radius: `rounded-sm`
- Placeholder колір: `#9C8A7E`

### Рейтинг (зірки)
- Заповнені: `#C8A86B` (золото)
- Порожні: `#E8DDD4`

## Зображення та фото

**Стиль:** Природне денне світло, теплі тони, мінімальний фон (білий / крафт)  
**Заборонено:** Синюваті холодні фільтри, темрява, захаращений фон  
**Кут зйомки:** Переважно ¾ зверху або фронт  
**Пропорції:** 4:3 для каталогу, 16:9 для hero, 1:1 для відгуків  
**Плейсхолдер (dev):** `bg-[#F5F0EB]` + SVG-іконка посуду

## Голос та тон (copywriting)

**Заголовки:** Короткі, конкретна вигода. «Посуд, що служить роками» > «Наш асортимент»  
**Body-текст:** Говори «ви», а не «ти». Уникай пасиву.  
**CTA:** Дієслово + вигода. «Купити зі знижкою», «Переглянути каталог»  
**Соціальний доказ:** Конкретні цифри. «2 847 клієнтів» > «тисячі клієнтів»  
**Заборонено:** «Найкращий», «Унікальний», «Революційний» — банальщина

## Специфіка для посуду (category best practices)

1. **Показуй контекст:** фото посуду з їжею / сервірований стіл продає краще за фото на білому тлі
2. **Матеріал — ключова деталь:** завжди вказуй (порцеляна, кераміка, скло, нержавійка)
3. **Догляд:** "можна в посудомийну машину" — сильний аргумент для UА-ринку
4. **Об'єм / розмір:** в мл або см на картці — зменшує повернення
5. **Комплект vs одиниця:** показуй обидва варіанти з різницею в ціні
6. **Уpsell:** натяк на набір («Доповни колекцію») на сторінці товару / кошику
7. **Сезонність:** активно використовуй свята (НГ, 8 березня, день народження)

## Структура лендінгу (conversion-oriented)

```
1. Header          — логотип + пошук + кошик + переваги-смужка
2. Hero + УТП      — 70% ваги конверсії
3. Цифри довіри    — швидке підтвердження соціального доказу
4. Каталог + фільтри — основна цінність, фільтрація без перезавантаження
5. Відгуки         — подолання заперечень до вибору
6. Гарантії        — подолання заперечень до покупки
7. Кошик (slide-out) — конверсія + upsell
8. FAQ             — підчищення останніх сумнівів
9. Lead capture    — для тих, хто ще не готовий
10. Footer         — довіра, контакти, юридика
```

**Правило черговості:** Довіра → Цінність → Дія → Заперечення → Лід

## Заборонено

- Автовідтворення відео зі звуком
- Попапи на вході (перші 30 сек)
- Більше 2 CTA одного типу в одному екрані
- Мигаючі елементи
- Стокові фото людей з ненатуральними посмішками
- Дрібний шрифт у юридичних текстах менше 12px
